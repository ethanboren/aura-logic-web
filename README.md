# Aura Logic - Predictive Financial Intelligence

AI-powered decisional engine that transforms massive unstructured data streams into actionable financial signals.

**Early Stage** - Currently building. Early access coming soon.

## Quick Start

1. Clone this repo
2. Open index.html in your browser - no build step needed

## Adding Images

The site has 2 image placeholders. Create an `images/` folder next to index.html.

### Image 1 - About Section (`images/about.jpg`)

Search on unsplash.com or pexels.com for: "tech startup team working dark office computer screens"

Pick a photo of people working at computers in a modern/dark-toned office. Recommended: 800x600px.

In index.html, find the comment `IMAGE #1`. Delete the `img-placeholder` div and uncomment the `<img>` tag above it.

### Image 2 - Dashboard Preview (`images/dashboard.jpg`)

Search for: "dark fintech dashboard UI data analytics charts"

Pick a dark-themed analytics dashboard screenshot. Recommended: 1920x1080px.

Same process: find `IMAGE #2`, delete placeholder, uncomment img tag.

## Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to Settings > Pages
3. Under Source, select main branch and root
4. Your site will be live

## Customize

- Email: Replace `hello@auralogic.ai` (appears 3 times)
- Metrics: Edit the hero numbers
- Roadmap: Update the timeline dates
- Colors: Edit CSS variables at the top of the style block

## Tech

Single HTML file. No build tools. Google Fonts + CSS animations + vanilla JS.
